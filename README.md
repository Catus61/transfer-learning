# Transfer Learning

## Dependencies

- python >= 3.5
- keras
- matplotlib
- numpy
- tensorflow

## Installation

```bash
pip install -e .
```

## Usage

```bash
transfer-learn -l fc2 -s 2 -e 1 -o output.hdf5
```

## Help

```bash
transfer-learn -h
```
